Tooltips

Use the ngxStrapTooltip directive to create Bootstrap tooltips with strings or templates.

Basic Usage

Import the PopupModule. This module exports the tooltip directive.

```typescript // app.module.ts // ... import {{ '{' }} PopupModule } from '@nowzoo/ngx-strap'; @NgModule({{ '{' }} imports: [ //... PopupModule ] }) export class AppModule {{ '{' }} } ```

Create a tooltip...

```html <button type="button" class="btn btn-primary" ngxStraptooltip title="tooltip Title">tooltip</button> ```

API

  • Selector: '[ngxStraptooltip]'
  • Export As: 'ngxStraptooltip'

Inputs

Title and Content
  • tooltipTitle: string | TemplateRef<any>
    Optional. Pass a string or a template for the tooltip title. If you don’t need a dynamic title use the title or data-title attribute.
  • tooltipContent: string | TemplateRef<any>
    Optional. Pass a string or a template for the tooltip content. If you don’t need dynamic content use the or data-content attribute.
Other Bootstrap tooltip Options

All the other Bootstrap tooltip options except selector have analogous directive inputs. Note that the following options can only be set once, when the directive is created.

  • tooltipAnimation: boolean
    Optional. You can also use the data-animation attribute.
  • tooltipDelay: number | {{ '{' }}show: number, hide: number}
    Optional. You can also use the data-delay attribute.
  • tooltipHtml: boolean
    Optional. You can also use the data-html attribute. Note that if you use the tooltipTitle input, HTML is always enabled.
  • tooltipContainer: string | HTMLElement | false
    Optional. You can also use the data-container attribute.
  • tooltipPlacement: string | ((popupEl: HTMLElement, triggerEl: HTMLElement) => string)
    Optional. You can also use the data-placement attribute.
  • tooltipTemplate: string
    Optional. You can also use the data-template attribute.
  • tooltipTrigger: string
    Optional. You can also use the data-trigger attribute.
  • tooltipOffset: number | string
    Optional. You can also use the data-offset attribute.
  • tooltipFallbackPlacement: string | string[]
    Optional. You can also use the data-fallbackPlacement attribute.
  • tooltipBoundary: string | HTMLElement
    Optional. You can also use the data-boundary attribute.
Other Inputs
  • tooltipEnabled: boolean
    Optional. Default true. Enable/disable the tooltip.
  • tooltipDismissOnClickOutside: boolean
    Optional. Use this to hide the tooltip when the user clicks or focuses anywhere outside the tooltip itself.

Outputs

  • events: EventEmitter<Event>
    The native Bootstrap tooltip events.

Functions

  • show(): void
    Show the tooltip manually.
  • hide(): void
    Hide the tooltip manually.
  • toggle(): void
    Toggle the visibility of the tooltip manually.
