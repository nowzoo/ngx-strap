Popovers Using Dynamic Strings

Use the popoverTitle and popoverContent inputs with dynamic strings to set the popover title and content.

```html <button type="button" class="btn btn-primary" ngxStrapPopover [popoverContent]="'The current time is ' + time" [popoverTitle]="'Open for ' + popoverOpenFor + ' second(s)'" (events)="onPopoverEvent($event)"> Popover with Dynamic Strings </button> ```
