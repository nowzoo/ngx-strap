Dismiss on Click Outside

Use [tooltipDismissOnClickOutside]="true" to dismiss the tooltip when the user clicks or focuses anywhere other than on the tooltip itself.

Link
```html <button type="button" class="btn btn-primary mb-2" ngxStrapTooltip data-trigger="click" [tooltipDismissOnClickOutside]="true" [tooltipTitle]="myTemplate"> tooltipDismissOnClickOutside: true </button> ```
Link to click.

Interacting with these elements will not dismiss the tooltip.
