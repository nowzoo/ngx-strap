Dropdowns

Use the ngxStrapDropdown directive to listen for dropdown events.

Note using this directive is entirely optional. You can create dropdowns with just the native bootstrap data- attributes and classes. Only use the directive if you need access to the dropdown events.

API

  • selector: [ngxStrapDropdown]
  • exportAs: ngxStrapDropdown
Inputs

None

Outputs
  • events: EventEmitter<Event> The native Bootstrap events.
Properties
  • status: 'hidden' | 'show' | 'shown' | 'hide' The current status of the dropdown (the last event's type.)

Demos

Basic Demo

Use the ngxStrapDropdown in the top-level .dropdown (or .dropup etc.) element. The only other required markup is:

  • a toggle element with data-toggle="dropdown", and
  • an element with the .dropdown-menu class.

The directive takes care of adding ids and aria- attributes if they are missing.

Dropdown Events and Status

The directive exposes the native Bootstrap dropdown events through the events output. The current status of the dropdown is available through the status property. This is one of 'hidden', 'show', 'shown' or 'hide'.
