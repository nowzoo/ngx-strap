Collapse

Directive: ngxStrapCollapse

API

  • selector: [ngxStrapCollapse]
  • exportAs: ngxStrapCollapse
Inputs
  • ngxStrapCollapse: boolean Whether the collapse should be shown.
Outputs
  • events: EventEmitter<Event> The native Bootstrap collapse events.
Properties
  • status: 'hidden' | 'show' | 'shown' | 'hide' The current status of the collapse (the last event's type.)

Demos

Basic Demo
Collapse Events and Status

The directive exposes the native Bootstrap collapse events through the events output. The current status of the collapse is available through the status property. This is one of 'hidden', 'show', 'shown' or 'hide'.
