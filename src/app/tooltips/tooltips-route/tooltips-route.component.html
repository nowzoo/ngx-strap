Tooltip with a template

Use [ngxStrapTemplateTooltip]="templateRef" to use an ng-template for the tooltip's content. This is useful when you want to interpolate variables.

Showing and hiding programmatically

Use data-trigger="manual" with either ngxStrapTooltip or ngxStrapTemplateTooltip to control the tooltip's visibility from your code or elsewhere in the markup. The show(), hide() and toggle() return promises that resolve when the tooltip is completely shown/hidden. You can track the tooltip's visibilty with the state property.

Disabling the tooltip

Use the directive's disable(), enable() and toggleEnabled() methods to control whether the tooltip will be shown. You can track this with the disabled property.

Tooltips go away

A tooltip (and the associated template in the case of ngxStrapTemplateTooltip) are disposed of when the directive is destroyed, such as on a route change.
