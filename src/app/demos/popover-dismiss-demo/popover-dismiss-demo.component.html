Dismiss on Click Outside

Use [popoverDismissOnClickOutside]="true" to dismiss the popover when the user clicks or focuses anywhere other than on the popover element itself.

Clicking or focusing elements outside this popover will hide it.
Link
```html <button type="button" class="btn btn-primary mb-2" ngxStrapPopover [popoverDismissOnClickOutside]="true" title="Dismiss on Click Outside: True" [popoverContent]="myTemplate"> Popover </button> ```

Link to click.

Interacting with these elements will not dismiss the popover.
