Show and Hide the Popover Manually

To manually control the popover's trigger behavior, set data-trigger="manual" and use the directive's show(), hide() and toggle() methods. The directive is exported as ngxStrapPopover.

```html <button type="button" class="btn btn-primary mr-2" ngxStrapPopover data-trigger="manual" data-placement="top" #popover="ngxStrapPopover" title="Popover Title" data-content="Some popover content."> Manual Popover </button> ```
