Popovers

Use the ngxStrapPopover directive to create Bootstrap popovers with strings or templates.

Basic Usage

Import the PopupModule. This module exports the popover directive.

```typescript // app.module.ts // ... import {{ '{' }} PopupModule } from '@nowzoo/ngx-strap'; @NgModule({{ '{' }} imports: [ //... PopupModule ] }) export class AppModule {{ '{' }} } ```

Create a popover...

```html <button type="button" class="btn btn-primary" ngxStrapPopover title="Title set with the title attribute" data-content="Content set with the data-content attribute.">Popover</button> ```

API

  • Selector: '[ngxStrapPopover]'
  • Export As: 'ngxStrapPopover'

Inputs

Title and Content
  • popoverTitle: string | TemplateRef<any>
    Optional. Pass a string or a template for the popover title. If you don’t need a dynamic title use the title or data-title attribute.
  • popoverContent: string | TemplateRef<any>
    Optional. Pass a string or a template for the popover content. If you don’t need dynamic content use the or data-content attribute.
Other Bootstrap Popover Options

All the other Bootstrap popover options except selector have analogous directive inputs. Note that the following options can only be set once, when the directive is created.

  • popoverAnimation: boolean
    Optional. You can also use the data-animation attribute.
  • popoverDelay: number | {{ '{' }}show: number, hide: number}
    Optional. You can also use the data-delay attribute.
  • popoverHtml: boolean
    Optional. You can also use the data-html attribute. Note that if you use the popoverTitle or popoverContent inputs, HTML is always enabled.
  • popoverContainer: string | HTMLElement | false
    Optional. You can also use the data-container attribute.
  • popoverPlacement: string | ((popupEl: HTMLElement, triggerEl: HTMLElement) => string)
    Optional. You can also use the data-placement attribute.
  • popoverTemplate: string
    Optional. You can also use the data-template attribute.
  • popoverTrigger: string
    Optional. You can also use the data-trigger attribute.
  • popoverOffset: number | string
    Optional. You can also use the data-offset attribute.
  • popoverFallbackPlacement: string | string[]
    Optional. You can also use the data-fallbackPlacement attribute.
  • popoverBoundary: string | HTMLElement
    Optional. You can also use the data-boundary attribute.
Other Inputs
  • popoverEnabled: boolean
    Optional. Default true. Enable/disable the popover.
  • popoverDismissOnClickOutside: boolean
    Optional. Use this to hide the popover when the user clicks or focuses anywhere outside the popover itself.

Outputs

  • events: EventEmitter<Event>
    The native Bootstrap popover events.

Functions

  • show(): void
    Show the popover manually.
  • hide(): void
    Hide the popover manually.
  • toggle(): void
    Toggle the visibility of the popover manually.
