Show and Hide the Tooltip Manually

To manually control the tooltip's trigger behavior, set data-trigger="manual" and use the directive's show(), hide() and toggle() methods. The directive is exported as ngxStrapTooltip.

```html <button type="button" class="btn btn-primary mr-2" ngxStrapTooltip data-trigger="manual" #tooltip="ngxStrapTooltip" title="Toolitip Title"> Manual Tooltip </button> ```
