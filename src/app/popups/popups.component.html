Tooltips and Popovers

Directives for creating Bootstrap tooltips and popovers from plain or dynamic strings, or templates.

Basic Demo

The ngxStrapTooltip and ngxStrapPopover directives work out of the box with Bootstrap's native data- attributes.

Dynamic Strings

Use [tooltipTitle], [popoverTitle] and [popoverContent] to pass dynamic strings.

Templates

For more complex markup and bindings, pass template references to [tooltipTitle], [popoverTitle] and [popoverContent].

Enabling / Disabling

Use [tooltipEnabled] to control whether a tooltip is shown. Use [popoverEnabled] to control whether a popover is shown.

Dismiss on Outside Click or Focus

Use [tooltipDismissOnClickOutside]="true" or [popoverDismissOnClickOutside]="true" to control whether the popup is dismissed when the user focuses or clicks an element outside of it.

Setting Options

Most popover and tooltip options can be set using data- attributes. You may, however, find it convenient to pass options to the directive directly. Use [tooltipOptions]="..." or [popoverOptions]="...". Both accept an instance of IPopupOptions.

Events

The native Bootstrap tooltip and popover events are exposed via tooltipEvents and popoverEvents respectively.

Manual Trigger

Both the tooltip and popover directives have show(), hide() and toggle() methods.
