No Animation
Disable animation by removing the .fade class from the .modal element.
Small Modal
Add the .modal-sm class to the .modal-dialog element.
Large Modal
Add the .modal-lg class to the .modal-dialog element.
Centered Modal
Add the .modal-dialog-centered class to the .modal-dialog element.
No Backdrop Modal
Set data-backdrop="false" class in the top-level .modal element.
Static Backdrop

Clicking on the backdrop does not dismiss the modal.

Set data-backdrop="static" class in the top-level .modal element.
Static Backdrop

Pressing the esc key does not dismiss the modal.

Set data-keyboard="false" class in the top-level .modal element.
Static Backdrop

The modal does not receive focus when shown.

Set data-focus="false" class in the top-level .modal element.
