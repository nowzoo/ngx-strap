Modals

Use the ModalService to display Bootstrap modals from templates.

Basic Usage

Import the ModalModule and call ModalModule.forRoot().

```typescript // app.module.ts // ... import {{ '{' }} ModalModule } from '@nowzoo/ngx-strap'; @NgModule({{ '{' }} imports: [ //... ModalModule.forRoot() ] }) export class AppModule {{ '{' }} } ```

Inject ModalService in your component.

```typescript import {{ '{' }} ModalService } from '@nowzoo/ngx-strap'; // ... export class ModalMinimalDemoComponent {{ '{' }} constructor( public modalService: ModalService ) {{ '{' }} } } ```

API

Service: ModalService

  • show(templateRef: TemplateRef<any>, opts?: IModalOptions): IModalInstance
    Show a modal based on a template.

Interface: IModalInstance

  • events: EventEmitter<Event>
    The native Bootstrap modal events.
  • shown: Promise<void>
    Promise that resolves when the modal has been completely shown.
  • hidden: Promise<void>
    Promise that resolves when the modal has been completely hidden.
  • hide: () => Promise<void>
    Method to hide the modal instance.
  • handleUpdate: () => Promise<void>
    Method to readjust the modalâ€™s position if the height of a modal changes while it is open.
