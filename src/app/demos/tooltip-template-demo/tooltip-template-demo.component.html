Tooltip Using a Template

For more complex html and bindings, use the tooltipTitle input with template reference.

{{time}}
```html <button type="button" class="btn btn-primary" ngxStrapTooltip [tooltipTitle]="myTemplate"> Tooltip from Template </button> <ng-template #myTemplate> <i class="fas fa-spin fa-spinner fa-fw"></i> {{ '{{' }}time}} </ng-template> ```
