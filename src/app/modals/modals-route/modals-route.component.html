Modals

A simple directive to apply to an ng-template containing some Bootstrap modal markup.

Docs | Library Code

Minimal Demo

There are no options or inputs. Control the modal behavior with native Bootstrap modal classes and data- attributes, in your markup. Use the directive's show() method to show the contents of an ng-template as a modal.

Promises

The show() method returns an instance of IModalInstance, with shown and hidden promises.
