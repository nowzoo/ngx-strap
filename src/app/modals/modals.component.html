Modals

Use the ngxStrapModal directive with an ng-template to create a Bootstrap modal.

Basic Usage

Import the ModalModule. This module exports the modal directive.

```typescript // app.module.ts // ... import {{ '{' }} ModalModule } from '@nowzoo/ngx-strap'; @NgModule({{ '{' }} imports: [ //... ModalModule ] }) export class AppModule {{ '{' }} } ```

Create a modal with an ng-template and the [ngxStrapModal] selector.

```html <ng-template ngxStrapModal #modal="ngxStrapModal"> <!-- Complete Bootstrap modal markup here --> </ng-template> <button (click)="modal.show()">Show Modal</button> ```

Directive API

  • Selector: [ngxStrapModal]
  • Export As: ngxStrapModal

Inputs

None. All Bootstrap modal options can be set in the markup you place inside the ng-template, via the native Bootstrap data- attributes and classes.

  • Bootstrap modal options (except show) can be set via data- attributes in the top-level <div class="modal">.
    • backdrop: data-backdrop
    • keyboard: data-keyboard
    • focus: data-focus
  • Animation is controlled by adding or omitting the fade class in the top-level <div class="modal">.
  • Modal size is controlled by adding the modal-lg or modal-sm classes to the <div class="modal-dialog"> element.
  • You can center the dialog vertically in the viewport by adding adding the modal-dialog-centered class to the <div class="modal-dialog"> element.

Outputs

  • events: EventEmitter<Event>
    The native Bootstrap modal events.

Methods

  • show(): {{ '{' }}shown: Promise<void>, hidden: Promise<void>}
    Shows the modal. The shown and hidden promises resolve when the modal is completely shown or hidden (i.e. after the animation completes.)
  • hide(): <void>
    Hides the modal.
  • handleUpdate(): <void>
    Use this when the height of the modal content changes. See the Bootstrap docs.
