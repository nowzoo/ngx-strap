Collapse

Bootstrap's implementation of collapse ties an element's visibility to a button or link that toggles it. The ngxStrapCollapse directive shows or hides an element, based on a boolean input, without reference to any other element.

Documentation

Basic Demo

Use [ngxStrapCollapse]="myVar" to create a collapsible element whose visibility is tied to myVar. The directive adds the .collapse class if it's missing.

Events

The directive exposes the native Bootstrap collapse events through the events output.
